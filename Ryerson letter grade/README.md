## Ryerson letter grade
Given the percentage grade, calculate and return the letter grade that would appear in
the Ryerson grades transcript.

| pct | Expected result |  
| :--- | :--- |
| 45 | 'F' |
| 62 | 'C-' |
| 89 | 'A' |
| 107 | 'A+' |
